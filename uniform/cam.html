Simple HTML Form with Angular

This is a basic HTML form, with basic angular added. There is no submit, only a form and a displaying of values stored in that form.

  • The top HTML tag has: ng-app="" ... to identify this as an angular infused app.
  • There is a script tag in the HEAD element to access the AngularJS binaries.
  • Two different Input elements: text and radio.
  • There is no submit ie: input type="submit"
  • We name the form ("MyForm"), so Angular has access to that data.
  • We print out "MyForm" at the end, in JSON format.
  • The attribute "checked" doesn't work with Angular, use: ng-checked="true"
  • If we omit ng-model="..." then we do not have access to that input data. (See prov)
