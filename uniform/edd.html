HTML Form with Angular using ng-submit

This is a an Angularized HTML form, using ng-submit on the form to call an Angular function: onSubmit.

  • The top HTML tag has: ng-app="eddsApp" to identify this as an angular infused app.
  • There is a script tag in the HEAD element to access the AngularJS binaries.
  • There is a script tag with local angular code in the BODY element
  • In the script file, we create an angular module named: "eddsApp".
  • We need to set up an ng-controller to acess local angular code.
  • Two different Input elements: text and radio.
  • Each input element gets has: ng-model="MyModel.xxx" so Angular has access to it.
  • The button has a type of "submit" - so it calls ng-submit on the FORM tag.
  • We print out the Angular Model data at the end, in JSON format.
  • We print out the Form data at the end, in JSON format.
  • Request is sent over http, success and error return codes are handled.
