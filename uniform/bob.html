Simple HTML Form

This is a basic HTML form, it has:

  • A Form element with an "action" to perform something
  • The "action" uses "get", which is the default if omitted.
  • Three different Input elements: text, radio, and submit.
  • The "submit" will call the "action='doSomething.rb' defined in the form tag.
  • Currently the action does nothing, but you can see how it works.
