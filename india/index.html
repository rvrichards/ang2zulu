Input Form Goodness

We will use the following styling:

  • ng-valid -- green
  • ng-invalid -- yellow
  • ng-empty -- red
  • ng-dirty -- brown
  • ng-touched -- violet
  • ng-pending -- blue
  • ng-not-empty -- orange
  • ng-pristine -- lightblue
Note that the order above determines what gets styled.
